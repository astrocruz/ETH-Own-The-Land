# ETH-Own-The-Land

Own the Land is a web3 tool for people to search, verify, update, purchase and sell properties.

frontend link: https://github.com/esmemag96/ETH-Own-The-Land/tree/master/own-the-land

**Tech Stack**
- Vue.js
- Tailwind
- Web3.js
- Ether.js
- WalletConnect

**Contract**
- HardHat
- Solidity 
- ERC721
- Polygon
